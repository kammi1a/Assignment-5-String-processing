# StringProcessor

A Java class for processing strings. This class includes methods for password validation, digit and word counting, and basic arithmetic expression evaluation.

## Features

- **Password Strength Check**: Validates if a password is strong based on character types (uppercase, lowercase, digits, special characters).
- **Digit Counting**: Counts the number of digits in a given string.
- **Word Counting**: Counts the number of words in a given string.
- **Expression Evaluation**: Evaluates simple arithmetic expressions with addition and subtraction.
